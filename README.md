# Project Thrishul

Project Thrishul is a Fedora-based Linux distribution designed for cybersecurity analysts, threat hunters, DFIR professionals, SOC engineers, and security researchers.

## Vision

Provide a secure, lightweight, analyst-focused Linux distribution with modern desktop technologies and practical cybersecurity tooling.

## Core Goals

- Fedora-based
- Lightweight
- Secure by default
- Hyprland desktop experience
- Easy NVIDIA support
- DNF package management
- RPM Fusion integration
- COPR repository support
- Btrfs snapshot support
- Analyst-focused tooling
