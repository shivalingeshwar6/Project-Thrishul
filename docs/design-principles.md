# Design Principles

## Minimal by Default

Install only what users need.

## Secure by Default

Enable:

- SELinux
- Firewalld

## Analyst Focused

Tools should support:

- Monitoring
- Detection
- Investigation
- Forensics

## Fedora Compatible

Maintain compatibility with Fedora repositories and tooling.

## User Friendly

Provide simple methods for:

- NVIDIA installation
- Repository management
- Snapshot management
