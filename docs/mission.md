# Mission Statement

Project Thrishul exists to provide a Linux distribution specifically designed for cybersecurity analysts.

Unlike traditional security distributions focused primarily on offensive security and penetration testing, Project Thrishul prioritizes:

- Threat Hunting
- Security Monitoring
- Digital Forensics
- Incident Response
- Detection Engineering
- Security Research

## Target Users

- SOC Analysts
- Threat Hunters
- DFIR Analysts
- Security Researchers
- Blue Team Professionals
- Security Engineers

## Principles

### Security First

Security features should be enabled by default.

### Lightweight

Avoid unnecessary packages and services.

### Stability

Prioritize reliability over unnecessary complexity.

### Productivity

Tools should help analysts work efficiently.

### Transparency

Use open-source software whenever possible.
